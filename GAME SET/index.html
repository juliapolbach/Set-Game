Set Game!

Inspired in the 70's Set card game, this renewed version of SET GAME challenge your visual acuity and tone up your pattern recognition.

To claim SET you must find three cards that satisfy ALL of these conditions.

Instructions :

They all have the same color, or they have
3 different colors.

They all have the same size, or they have
3 different sizes.

They all have the same fill, or they have
3 different fills.

They all have the same border, or they have
3 different borders.

Score : pt
